# Train timetable & ticket service
Term project for programming technologies discipline.

## Current features
- Test health endpoint
- JWT Authorization;
- Database persistancy;
- Check timetable and tickets;
- Administrator capabilities for:
    - timetable operator;
    - customer support;

## Future features
- Token refreshing;
- Ticket lock ability;
- Ticket buying (through mock payment gateway);
- Report generation;
- Unit and functional tests.
